'UEFA.com' site parsing

Script for parsing 'uefa.com' site to obtain cool web-calendars with UEFA tournament events. Those calendars maintan two types of events:

  • "whole day" event for each tournament stage with list of matches
  • separate event for each match with information about participants (clubs), stadium, scores (for completed games).

Event titles contain symbols ‚öΩ & üèÜ, so you can easily identify what the event is. Also, description of each match contains logos of playing teams.

Script: uefa_calendar.py Download

Usage:

import uefa_calendar as uefa

uefa.export_calendar(r'champions_league.ics', uefa.CHAMPIONS_LEAGUE)
uefa.update_score(r'europa_round6.ics', uefa.EUROPA_LEAGUE, 6, None)
